# Electropower website draft

A modern, responsive single-page website for Electropower Engineering.

Open `index.html` in a browser for a quick preview, or serve this folder with any static web server.

## Before launch

- Confirm all wording and services with Electropower.
- Confirm the current telephone number and postal address.
- Add an email address if required.
- Add privacy/cookie information if analytics or contact forms are introduced.
